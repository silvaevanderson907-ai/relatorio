Painel simples de controle de vendas de carros por mês.
Arquivos:
- index.html: interface web leve para inserir vendas e ver relatório mensal.
- styles.css: estilos mínimos.
- app.js: lógica local (armazenamento em localStorage) e cálculo de totais.

Instruções: Abra sales/index.html no navegador para usar o controle de vendas.
